# ResearchWeaver AI — UI Guidelines

## Design Philosophy

ResearchWeaver AI should feel like a premium research operating system rather than a chatbot.

### Principles

- Clean and spacious layouts
- Editorial typography
- Soft shadows
- Rounded corners
- Minimal color palette
- Smooth micro-interactions
- Consistent spacing using an 8px grid
- Accessibility-first components

### Inspiration

- Linear
- Notion
- Vercel
- Arc Browser
- Raycast

### Avoid

- Neon gradients
- Glassmorphism overload
- Bootstrap dashboard aesthetics
- ChatGPT clone layouts
- Cluttered interfaces

### Core Experience

The Knowledge Graph is the center of the product.

The AI Chat is an assistant—not the main feature.

Every feature should reinforce the workflow:

Upload → Understand → Discover → Plan → Explore
